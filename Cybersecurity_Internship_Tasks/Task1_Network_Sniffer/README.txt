Project: Network Sniffer

Description:
This project is a Python-based network sniffer built using Scapy. It captures live network packets and analyzes them.

Features:
- Captures real-time network traffic
- Displays source and destination IP addresses
- Identifies protocols (TCP/UDP)
- Extracts port numbers
- Logs packet data into a file

Requirements:
- Python 3.12
- Scapy
- Npcap (for Windows)

How to Run:
1. Open CMD as Administrator
2. Run: python sniffer.py
