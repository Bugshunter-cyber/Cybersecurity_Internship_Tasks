Project: Keylogger (Educational Purpose)

Description:
This project simulates a basic keylogger in a controlled environment to understand how keystroke logging works.

Features:
- Captures keystrokes
- Saves them into a file
- Demonstrates security risks

Risks:
- Password theft
- Data leakage
- Privacy invasion

Prevention:
- Use antivirus software
- Avoid untrusted downloads
- Monitor system activity

How to Run:
1. Open CMD
2. Run: python keylogger.py
